# Overview

The Custom Header plugin for Movable Type provides the ability for
designers to specify with their template sets constraints for a 
custom header that this plugin will then allow users to easily
replace with an image uploaded from their local computer.

# Installation

To install this plugin follow the instructions found here:

    http://tinyurl.com/easy-plugin-install
